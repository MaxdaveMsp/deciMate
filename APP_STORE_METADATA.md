## Description draft
DeciMate helps you monitor sound levels during rehearsals, DJ sets, small events, and live-sound work. It turns your iPhone into a practical SPL monitoring companion with a clean meter, calibration offset, threshold alerts, session logs, CSV export, and expressive reactions that make level changes easy to understand at a glance.

Features:
- Live approximate SPL display
- Manual calibration offset against a reference meter
- Warning, critical, and peak thresholds
- Fast/slow response options
- Session graph and CSV export
- Friendly companion reactions
- deciMate Live Link preview for local network workflows

Important: DeciMate uses the device microphone for practical monitoring. It is not a certified Class 1 or Class 2 sound level meter and should not be used as the sole tool for legal/compliance measurements.

## Privacy summary
- Microphone: used for sound-level estimation.
- Local network: used only when Live Link is explicitly enabled.
- Data storage: session data is local to the device unless the user exports it.
- Tracking: none planned for MVP.
- Accounts: none planned for MVP.
