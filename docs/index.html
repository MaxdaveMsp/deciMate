Features

Built for the real world,
not a lab.

Designed for sound engineers, DJs, teachers, venue managers, and anyone who needs a fast, readable picture of what's happening with sound levels.

🎚️

Live Arc SPL Meter

A 300° arc meter fills in real time as levels rise — cyan for safe, shifting through yellow and orange to red at peak. Average and session peak always visible.

🤖

Expressive Mascot

A tiny audio-native mascot with white oval eyes reacts to every state change — calm waveform smile when quiet, panicked × eyes at peak events. Spring animations, random blinks, EQ bar ears.

⚡

Smart Thresholds

Warning, Critical, and Peak thresholds shown as tick marks on the arc. The whole UI — mascot, chart, metric pills — shifts color as you cross each one.

📋

Exposure Reports

Every export includes Leq, OSHA dose %, NIOSH dose %, 8-hour TWA, and EU LEX,8h — everything needed for practical noise exposure assessment.

🎯

Calibration Offset

Apply a manual dB offset using a reference meter or acoustic calibrator. "Match current reading" button sets it in one tap.

📊

Session History Chart

A live area + line chart shows the last 90 samples. Color follows state — green baseline, yellow when worried, red at peak. Exported CSV includes every sample.

🔗

deciMate Live Link

Optional local-network HTTP server streams a live JSON snapshot of current SPL, average, peak, and state. Poll from any dashboard, script, or HTTP tool on the same Wi-Fi.

🔒

Privacy First

No account. No analytics. No cloud. Audio is processed on-device and never recorded or transmitted. Session data stays local until you choose to export it.

Noise Exposure Reports

Export a real report,
not just numbers.

Every session export includes the calculations that actually matter for noise exposure assessment — not just a raw dB log.

OSHA
29 CFR 1910.95 · 5 dB exchange rate
US Standard
Criterion level90 dBA / 8 hrs
Action level85 dBA TWA (50% dose)
PEL90 dBA TWA (100% dose)
Report shows dose %, 8-hr TWA, and compliance status
NIOSH
REL · 3 dB exchange rate · more protective
Recommended
Criterion level85 dBA / 8 hrs
Action level85 dBA TWA (50% dose)
REL85 dBA TWA (100% dose)
Stricter 3 dB exchange rate — favoured by occupational health orgs
EU Directive 2003/10/EC
Noise at Work Directive
EU Standard
Lower action valueLEX,8h = 80 dB
Upper action valueLEX,8h = 85 dB
Exposure limitLEX,8h = 87 dB
Report normalises Leq to 8-hour working day (LEX,8h)
Leq — Equivalent Level
Energy-averaged SPL · the correct dB average
Key Metric
Leq is computed using the energy-averaging formula 10·log₁₀(Σ10^(Lᵢ/10)/N) — the internationally accepted method for noise exposure. Not a simple arithmetic average.
