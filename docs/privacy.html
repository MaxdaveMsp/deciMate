1. Who we are

DeciMate SPL is developed by David Torres / Loud and Beyond. This privacy policy explains what data the app accesses, how it is handled, and what — if anything — ever leaves your device.

Contact: info@loudandbeyond.com

2. Microphone access

DeciMate SPL requests access to your device microphone solely to measure sound pressure levels. No audio is ever recorded, stored, or transmitted.

The app uses Apple's AVAudioEngine to tap the microphone input. For each 4096-sample buffer, the RMS (root mean square) value is computed and immediately converted to a dB reading. The raw audio samples are discarded instantly — they never touch storage, network, or any buffer outside the calculation.

3. Data stored on your device

The following data is stored locally on your iPhone only, using Apple's UserDefaults and in-memory session storage:

  • Calibration offset — a single dB number, saved in UserDefaults
  • Threshold settings — warning, critical, and peak dB values
  • Measurement weighting and response preferences
  • Active session samples — held in memory only, not written to disk unless you export

None of this data is transmitted to any server, shared with any third party, or synced to iCloud.

4. Data you choose to export

Tapping Export generates a noise exposure report on your device. The file is presented via the standard iOS share sheet — you control exactly where it goes (Files, Mail, AirDrop, etc.). DeciMate SPL does not receive, store, or transmit a copy of the exported file.

The exported report includes: session timestamps, SPL samples, peak readings, threshold states, Leq, OSHA/NIOSH dose calculations, and EU LEX,8h. It contains no device identifiers, location data, or personal information.

5. DeciMate SPL Live Link (optional feature)

Live Link is an opt-in feature that you must explicitly start by tapping "Start Live Link" in Settings. When active, it can open a local HTTP server on port 8080 for JSON status and live stream endpoints, and can optionally send OSC messages to a host and port you configure.

  • Only runs when you actively start it — it is completely off by default
  • Only accessible to devices on the same local Wi-Fi network
  • Not accessible over the internet
  • OSC is only sent when you turn on Stream OSC and choose a destination
  • Does not log, store, or forward any requests or data
  • Stops completely when you tap "Stop Live Link" or close the app

This feature requires iOS to grant local network access. The permission prompt will appear the first time you start Live Link.

6. Analytics and tracking

DeciMate SPL contains no analytics framework, no advertising SDK, and no tracking code of any kind. We do not collect:

  • Usage statistics or event tracking
  • Crash reports (other than those Apple collects if you opt in to sharing with developers)
  • Device identifiers (IDFA, IDFV, or any other ID)
  • Location data
  • Any personal information

7. Third-party services and SDKs

DeciMate SPL uses no third-party libraries, SDKs, or APIs. The entire app is built using Apple's own frameworks:

  • SwiftUI — user interface
  • AVFoundation — microphone access and audio processing
  • Combine — reactive data flow
  • Network framework — Live Link local HTTP server and optional OSC UDP output
  • Charts — session history graph
  • UniformTypeIdentifiers — file export

8. Children's privacy

DeciMate SPL does not knowingly collect any information from children under 13. The app contains no advertising, in-app purchases targeting children, or any features designed to collect data from minors.

9. Apple's data collection

If you have opted in to sharing crash and usage data with app developers (via iOS Settings → Privacy → Analytics & Improvements), Apple may provide us with anonymised crash logs. This is controlled entirely by your iOS settings and Apple's own privacy policy — we cannot change what Apple collects on your behalf.
