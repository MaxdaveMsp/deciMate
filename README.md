# deciMate

Friendly SPL monitoring, logging, reporting, and companion reactions for live sound.

## Product split

### deciMate Basic
- Approximate iPhone mic SPL monitoring
- Manual calibration offset
- Warning / critical / peak thresholds
- Companion reactions
- Session logging
- CSV export

### deciMate Pro
- Advanced reports
- Multiple calibration profiles
- External mic profiles
- Custom companion behavior
- deciMate Live Link: local HTTP/WebSocket/OSC bridge for SMAART, RiTA, Max/MSP, Q-SYS, TouchDesigner, OBS, Node-RED, and dashboards

## Measurement caveat
Phone microphones are useful for practical monitoring but are not certified Class 1/Class 2 measurement microphones. Product wording should avoid certified/compliance claims unless validated with calibrated external hardware.

## Current status
